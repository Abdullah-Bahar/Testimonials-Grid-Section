### Kullanılan Teknolojiler

- Anlamlı ve düzenli HTML5 yapısı
- Modern CSS ile tasarım 
- Flexbox - Kart başlıkları için
- CSS Grid - Ana layout `grid-template-areas` ile kuruldu
- desktop-first Yaklaşımı
- Responsive Tasarım
